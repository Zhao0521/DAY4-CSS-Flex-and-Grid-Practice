### 學習心得
 - 不得不說，RWD真的有進階到！一邊要顧版面、一邊想是要用grid還是flex，又要兼顧美感，怕排版跑掉。
 - 不同裝置需要重新設置不同參數，尚未加入變數就已經寫著寫著自己就混亂了～
 - 目前想法是先打好設計圖稿，固定區域用grid areas暴力排版，小地方微調再用flex修正。
